# Instagram Profile Setup Skill

## Prerequisites
- `mlProfileId` from `pa-lookup.js <clientId> instagram`
- `credentials.username` and `credentials.password`
- `profileContent.bio` (150 chars max, generated by generate-bio.js with platform:instagram)

---

## Login

```bash
node /data/director/dispatch.js '{
  "mlProfileId": "<uuid>",
  "clientId": "<clientId>",
  "steps": [
    {"action": "goto", "url": "https://www.instagram.com/accounts/login/"},
    {"action": "wait", "ms": 2000},
    {"action": "click", "selector": "input[name=username]"},
    {"action": "type", "text": "<username>"},
    {"action": "click", "selector": "input[name=password]"},
    {"action": "type", "text": "<password>"},
    {"action": "click", "selector": "button[type=submit]"},
    {"action": "wait", "ms": 4000},
    {"action": "getPageText"}
  ]
}' --timeout 90
```

---

## Edit Profile

```bash
node /data/director/dispatch.js '{
  "mlProfileId": "<uuid>",
  "clientId": "<clientId>",
  "steps": [
    {"action": "goto", "url": "https://www.instagram.com/accounts/edit/"},
    {"action": "wait", "ms": 2000},
    {"action": "click", "selector": "input[name=fullName]"},
    {"action": "type", "text": "<display-name>"},
    {"action": "click", "selector": "textarea[name=biography]"},
    {"action": "type", "text": "<bio-150-chars>"},
    {"action": "click", "selector": "input[name=website]"},
    {"action": "type", "text": "<website-url-if-any>"},
    {"action": "click", "selector": "button[type=submit]"},
    {"action": "wait", "ms": 2000},
    {"action": "screenshot", "path": "/tmp/instagram-profile-done.png"},
    {"action": "getPageText"}
  ]
}' --timeout 120
```

## Completion Report Format

```
Instagram profile setup complete for [Client Name]:
- Display name: "[name]"
- Bio: "[bio text]" ([char count] chars)
- Website: [url or "not set"]
- MultiLogin X profile used: [profileId]
```
